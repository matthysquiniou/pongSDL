# Pong en C++ avec SDL

Ce projet fait partie de mes premiers programmes en **C++**.  
L’objectif principal était de découvrir la bibliothèque **SDL** et de l’utiliser pour créer un jeu simple de type **Pong**.

## But du projet
- Prendre en main la SDL (affichage, gestion des entrées clavier, boucle de jeu)
- Comprendre les bases d’un jeu en 2D
- Manipuler la logique de collision et de mouvement

## Contrôles

| Joueur | Touche | Action |
|--------|--------|--------|
| Paddle gauche | **Z** | Monter |
| Paddle gauche | **S** | Descendre |
| Paddle droit | **Flèche haut** | Monter |
| Paddle droit | **Flèche bas** | Descendre |

## Technologies utilisées
- C++
- SDL

## Remarque
Le projet est volontairement simple et sert surtout d’exercice d’apprentissage.
